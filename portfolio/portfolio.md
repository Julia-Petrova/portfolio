Here are a few samples of the technical documentation I produce. 
They can give you an idea of my style and documentation skills.
If you need more samples, let me know via [petrova751@gmail.com](mailto:petrova751@gmail.com)!

All the samples given here are based on a real system I documented in another language.
For use on this site, I have replaced the system's name with a fictional name, Datafuel, 
generated at [https://namelix.com](https://namelix.com).

The samples share the following features:
* **Audience:** Developers, system analysts, system architects, and deployment teams
* **Industries:** Data marts, data warehouses, software
* **Tools Used:** Markdown, Jekyll, Hydejack theme, and GitHub Pages
